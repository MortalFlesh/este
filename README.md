## Prerequisites

* [node.js](http://nodejs.org/) Node 8+
* [yarn](https://yarnpkg.com/)

## Starting a project

* `git clone https://github.com/este/este.git este`
* `cd este`
* `yarn`
* `cd backend`
* `yarn`
* `cd ..`
* `yarn dev`

* `yarn relay`
